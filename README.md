# AI 学习与实践记录

这是一个用于记录 AI 学习过程、实践项目和阶段总结的仓库。

内容会围绕大模型基础、RAG、Agent、LangGraph、MCP，以及综合项目实践展开。目标不是只停留在概念理解，而是通过持续编码、复盘和整理，形成一套可以运行、可以讲清楚、可以持续迭代的 AI 工程实践记录。

## 项目目标

- 系统学习 AI 应用开发相关的核心概念与工程方法
- 通过小型 Demo 和阶段项目沉淀可运行成果
- 逐步建立对 RAG、Agent、多 Agent 编排和 MCP 的完整理解
- 持续记录学习笔记、问题复盘和实践总结

## 学习主题

当前计划按照 4 个阶段推进：

1. 大模型基础与 RAG
2. Agent 核心概念与工具调用
3. 多 Agent、DAG 编排与 MCP
4. 综合项目实践与知识整理

## 重点方向

### 1. 大模型基础

- Transformer 基本机制
- Token、上下文窗口、推理流程
- Prompt 参数对结果的影响

### 2. RAG

- 文档加载与切片
- Embedding 与向量检索
- 重排序、混合检索、效果评估

### 3. Agent

- ReAct 思路与执行循环
- Function Calling / Tool Use
- Memory、Planning、可观测性

### 4. 多 Agent 与工作流

- DAG 编排
- Hierarchical 多 Agent 协作
- LangGraph 状态流转与条件分支

### 5. MCP

- MCP 的三层结构
- MCP Client / Server 基本通信方式
- 基于统一协议接入工具与数据源
